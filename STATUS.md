# 1. Current State

HBCE-OIS is published as an open technical pre-standard defining a deterministic operational identity layer for autonomous, human and hybrid systems.

The current release includes:

- core specification (HBCE-OIS v0.1)
- normative IPR schema
- compliance requirements
- architecture overview
- EU positioning note
- citation reference

The specification is designed to be replicable and independently implementable.

---

# 2. Implementation Readiness

HBCE-OIS is suitable for:

- experimental node deployment
- research and industrial testing
- AI and robotics integration testing
- append-only identity logging systems
- verification-layer prototyping

No centralized infrastructure is required.

Nodes may operate independently.
